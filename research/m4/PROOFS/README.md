# PROOFS

Human-readable proof writeups for M4 claims.

## Rules

- One file per lemma/theorem, e.g. `A_gap.md`, `B_order_reversal.md`.  
- State domain, quantifiers, and all DEFINITIONS defaults or deviations.  
- Mark gaps as `PROOF GAP`.  
- “By standard arguments” only if the standard theorem is named and hypotheses checked.  
- Simulation and “no counterexample found” are not proofs.
